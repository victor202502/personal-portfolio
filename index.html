Modern JavaFX Paint Application

This is a desktop paint application developed using Java with the JavaFX framework for the graphical user interface. It utilizes Maven for project build and dependency management. The application provides basic drawing functionalities including a pen tool, eraser, shape tools (rectangle, oval), color selection, and line width adjustment. Users can save their creations as PNG image files and load existing PNGs back into the canvas. The canvas supports zooming and panning for better navigation on larger drawings.

Requirements:
  • Requires Java Runtime Environment (JRE) version 18 or higher installed on the system.
  • Download the executable JAR file and run it from the command line using: java -jar YourAppName.jar (replace YourAppName.jar with the actual downloaded file name).

PHP Blog System

This is a dynamic blog application built from scratch using PHP for server-side logic and a PostgreSQL database (hosted on Neon) for data persistence. The application features user registration, login/logout functionality, and session management. Authenticated users can create, view, edit, and delete their own blog posts. All users can view posts from all authors and add comments to any post. Comment authors can also edit or delete their own comments. The project demonstrates CRUD operations, secure password hashing, user authentication, and basic database interaction with PDO.

Key Features:
  • User authentication (registration, login, logout) with password hashing.
  • CRUD operations for blog posts (Create, Read, Update, Delete) by post authors.
  • Public view of all posts with author information.
  • Commenting system on posts, with edit/delete capabilities for comment authors.
  • Responsive design using basic CSS, adaptable for various screen sizes.
  • Secure database interaction using PDO and prepared statements.
  • Deployed on Koyeb with an external Neon PostgreSQL database.
